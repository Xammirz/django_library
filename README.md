Настоящая библиотека Django. Есть возможность поиска книг по автору или названию книг с помощью google books API. Авторизация и Регистрация. Аренда книг. Есть админ и свои админские возможности как добовлять автора или убрать или изменить, та же история и с книгами. Информация о книгах и краткое описание их. ISBN для каждой книги свой уникальный. Админка тоже реализована поудобнее и понятней. Биография, годы жизни Авторов тоже присутствует
Образец django
Скелет (структура)
Python
Установите Python версии 3.6 и выше.

Зависмости
Установите все зависмости из файла requirements.txt

Запуск программы
Для запуска программы используйте команду 'python .\manage.py runserver'
